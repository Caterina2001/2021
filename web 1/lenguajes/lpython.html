PYTHON

Python fue creado a finales de los ochenta por Guido van Rossum y apareció oficialmente en el 1991. Es un lenguaje de programación interpretado y multiparadigma, ya que soporta parcialmente la orientación a objetos, programación imperativa y programación funcional. Así que podemos decir que es un lenguaje interpretado, dinámico y multiplataforma, super completo.

Los desarrolladores pueden usar en una amplia variedad de contextos. Aunque Python no haya sido creado específicamente para el desarrollo web, marcos como Django facilitan su uso como base para la programación web sin quitarle merito al Python antiguo donde se puede también escribir aplicaciones web, si se desea… sin problemas.

Es gratuito, no reporta gran dificultad para programar en cualquier sistema operativo, es dinámico y se utiliza como lenguaje de scripting. Además, hoy en día es ampliamente reconocido en el sector tecnológico porque diversos servicios que mencionare más adelante usan Python, al igual que muchos videojuegos incluyen en su programación parte de código escrito en Python, pero también sectores como el análisis Big Data y el desarrollo de la Inteligencia Artificial lo usan.

Entre sus numerosos Frameworks, puedo mencionar:

Django, que sería lo más cercano a Laravel en PHP o Ruby on Rails para Ruby. Un marco de trabajo completo y eficiente para desarrollar Aplicaciones Web de una gran complejidad con un mínimo esfuerzo.

Flask, tiene funcionalidades sencillas e inteligentes para construir cualquier sitio que se te pase por la cabeza.

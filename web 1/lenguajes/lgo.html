GO

Go es un lenguaje de programación compilado, concurrente, imperativo, estructurado, orientado a objetos desarrollado por Google que apareció oficialmente en el 2009.

Go es la solución ideal para aquellos que quieran programar aplicaciones Web y quieran algo mas sencillo que Java o alguna otra. Personalmente no la había escuchado, pero me doy cuenta que es un lenguaje que está en crecimiento y es altamente recomendado de estudiar en este 2021.

Es un lenguaje flexible y extensible con una sólida comunidad, facilita a los desarrolladores la creación de aplicaciones y se tienen ventajas si se usa para crear aplicaciones web de alto rendimiento. Es cierto que no es tan popular, pero está haciendo grandes pasos.

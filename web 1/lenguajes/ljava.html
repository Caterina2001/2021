JAVA

Fue comercializada por primera vez en 1995 por Sun Microsystems y actualmente es propiedad de Oracle. Java es un lenguaje de programación rápido, seguro, fiable y está en todas partes. Desde el 2012es uno de los lenguajes de programación más populares en uso, particularmente para aplicaciones de cliente-servidor de páginas web, permitiendo crear aplicaciones web complejas altamente escalables y seguras.

Java no fue diseñado para aplicaciones web. Pero si bien Java no se creó con el único propósito del desarrollo basado en navegador en mente, muchos aún lo consideran uno de los mejores lenguajes de programación web para aprender y usar. Es un lenguaje de propósito general capaz de crear con él todo tipo de proyectos y ejecutarlos en múltiples plataformas con mayores aplicaciones en el panorama actual. El hecho de ser un lenguaje multiplataforma nos da una ventaja significativa, pues hace unos años era necesario hacer un programa para cada sistema operativo, pero hoy en día no, porque se ha creado una máquina virtual de Java para cada plataforma, que hace de puente entre el sistema operativo y el programa llamada JVM, Java Virtual Machine.

Java es uno de los lenguajes de programación más utilizados en todo el mundo, manteniéndose siempre en el top 10 global de los mas utilizados en el desarrollo de aplicaciones Web profesionales y cada vez hay mas demanda en las empresas del mercado laboral. Es de suma importancia conocerlo y así en un futuro poder hasta utilizarlo en la creación de una pagina Web por su seguridad, fiabilidad y seguridad
